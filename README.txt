----------------------
     UNZIP FILE
----------------------

To unzip the file, right click on the folder. If on:

- MAC, click on 'Open'
- WINDOWS, click on 'Extract All'



----------------------
       MAKE FILE 
----------------------

-- Pre-conditions: 
Make sure all of the following files are located in the same directory (folder): 

card.cpp
card.h
deck.cpp
deck.h
player.cpp
player.h
go_fish.cpp

-- On command line:

run:

	$ make -f makefile


---------------------
     RUN PROGRAM
---------------------

-- On the command line:

run:

	$ ./gofish


----------------------
     CONDITIONS
----------------------

-- A text file, "gofish_results.txt", will be made in the folder and contain the results of the gameplay.
